English

A small tooling project for fetching NFT floor prices across multiple blockchains (ETH, Solana, ApeChain, etc.).
The script queries different providers (OpenSea, Moralis, Magic Eden / aggregators) depending on the chain, normalizes responses, and returns floor values in native currency + USD.
Goal: unified valuation logic for multi-chain NFT portfolios / dashboards.

Русский

Небольшой инструмент для получения floor-цен NFT на разных блокчейнах (ETH, Solana, ApeChain и т.д.).
Скрипт автоматически выбирает подходящий источник данных (OpenSea, Moralis, Magic Eden / агрегаторы) в зависимости от сети, нормализует формат ответов и возвращает floor в нативной валюте и в USD.
Цель — единая логика оценки мульти-чейн NFT портфеля / дашборда.
